# Unikron Real Estate & Design Platform

Unikron is a premium, minimalist web application for high-end construction and real estate. It uses AI to transform 2D floor plans into cinematic 3D interior experiences.

## Core Features
- **Design Studio**: Premium file upload for 2D floor plans.
- **Pricing Engine**: Dynamic project estimation based on styles, add-ons, and complexity.
- **Advanced Branding**: Integrated 3D metallic/glass logo system.
- **AI Monitoring**: Gemini-powered admin dashboard for real-time trend analysis.
- **Stripe Checkout**: Integrated project finalization payments.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS with Glassmorphism
- **AI**: Gemini (Agentic Monitoring), Nano Banana (3D Generation)
- **Payments**: Stripe

## Getting Started
1. Follow the [SETUP_GUIDE.md](./SETUP_GUIDE.md) to configure your environment.
2. Run `npm run dev` to start the local server.
